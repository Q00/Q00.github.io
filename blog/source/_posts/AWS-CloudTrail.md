# CloudTrail

<img src='../img/aws.png'>

CloudTrail 은  AWS 계정의 거버넌스, 규정 준수, 운영 및 위험 감사(risk auditing)를 활성화하도록 도와주는 AWS 서비스

- Per AWS account, per region, per service basis

- log는 아마존 s3 버킷에 저장됨
