#CloudWatch

AWS 클라우드 리소스 , AWS 어플리케이션 서비스 모니터링(performance monitoring)

  • 로그파일 수집, 모니터링
  • 시스템 전반의 가시성 확보
  • Dashboard를 통해 확인 가능
  • Alarm

  • CloudWatch Event는 AWS resource 상태 변화에 대한 대응에 도움됨

  • Standard Monitoring : 5 Minutes

  • Detail Monitoring : 1 Minute
